### Prerequisites

Clone the repo

```
git clone https://github.com/hendrikbgr/hackernews-clone && cd hackernews-clone
```

To install all needed requirements

```
npm install
```

## Running 🏃🏽‍♂️

Run the development server

```
npm run dev
```

Run the tests

```
npm run test
```
